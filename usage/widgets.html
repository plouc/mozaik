Configuration

Configure Mozaïk widgets

Widgets are the building blocks of a Mozaïk dashboard, a widget is in fact a react component which often communicates with an API.
Mozaïk comes with no widget, they are installed through extensions, see available extensions.

Widgets share some common properties which are all required:

keydescription
typethe type of widget to instantiate
xx position
yy position
columnswidth expressed in columns
rowsheight expressed in rows

for properties related to position/size see grid system.
