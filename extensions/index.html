Extensions

Extend Mozaïk with custom extensions
Mozaïk widgets are maintained as separate modules, available as mozaik-ext-<name> npm packages.
Below is a list of some already existing extensions. See widget configuration.
mozaik-ext-analytics

Google analytics widgets for Mozaïk.

google analytics
mozaik-ext-bamboo

Bamboo widgets for Mozaïk.

bamboo
mozaik-ext-calendar

Google calendar widgets for Mozaïk.

google calendar
mozaik-ext-github

Github widgets for Mozaïk.

github
mozaik-ext-heroku

Heroku widgets for Mozaïk.

heroku deploy
mozaik-ext-jenkins

Jenkins widgets for Mozaïk.

SauceLabs widgets for Mozaïk.

time

Travis widgets for Mozaïk

mozaik-ext-weather

Weather widgets for Mozaïk.
